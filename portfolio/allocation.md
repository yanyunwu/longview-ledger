# 资产配置计划

## 当前原则

- 价值投资为主，现金仓位用于等待安全边际。
- 单一股票初始仓位一般不超过 5%。
- 单一行业需要控制集中度。
- 消息面投机只能使用小仓位，并必须在交易记录中标记。

## 当前目标

| 模块 | 目标仓位 | 说明 |
|---|---:|---|
| A 股 | 0%-55% | 以长期价值和周期配置为主 |
| 港股 | 0%-20% | 等待后续纳入 |
| 美股 | 0%-20% | 等待后续纳入 |
| 贵金属 | 0%-20% | 黄金、白银跟踪 |
| 加密货币 | 0%-10% | 高波动资产，小仓位 |
| 现金/货基 | 20%-100% | 防守和等待机会 |
