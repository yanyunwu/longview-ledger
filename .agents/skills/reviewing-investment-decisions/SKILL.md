# Reviewing Investment Decisions

## Core Rule

Review against the original thesis and recorded facts, not hindsight explanations.

## Workflow

1. Read `AGENTS.md`.
2. Find the relevant asset dossier, transaction row, thesis, valuation note, and observation log.
3. Create or append a review under `reviews/trade_reviews/`, `reviews/thesis_reviews/`, or `reviews/monthly_reviews/`.
4. If the user identifies a mistake, append one row to `reviews/mistake_log.md`.
5. State: original judgment, actual result, what changed, whether the process was good, and what rule changes.
6. Avoid rewriting the old thesis or transaction. Add a dated update instead.

## Mistake Types

Prefer these tags:

- valuation error
- position sizing error
- thesis drift
- FOMO
- overtrading
- news overreaction
- failure to cut risk
- ignored cash flow
- ignored cycle risk
