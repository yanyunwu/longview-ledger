# 金融资产长期研究工作区

这个工作区用于长期跟踪金融资产研究、新闻、持仓操作和复盘。核心原则是事实、观点、操作分离：事实进入 CSV/JSON，研究判断进入 Markdown，真实交易进入 portfolio，错误和经验进入 reviews。

## 目录

- `assets/`：资产研究库，包含 A 股、港股、美股、贵金属、加密货币。
- `news/`：全球新闻、宏观、行业、公司新闻和每日摘要。
- `portfolio/`：持仓、交易流水、账户快照和资产配置计划。
- `reviews/`：交易复盘、投资逻辑复盘、月度复盘和错误日志。
- `templates/`：新增资产、观察日志、估值、交易和复盘模板。
- `data_raw/`：外部数据源原始缓存。
- `scripts/`：生成、检查脚本。
- `skills/`：本工作区专用的 Codex 协作技能。

## 使用纪律

- 默认以价值投资为主。
- 小仓投机必须在交易记录中标记为 `small_position_speculation`。
- 每次买入、卖出、加仓、减仓都要记录理由、预期结果和失效条件。
- 每次重大判断错误都要进入 `reviews/mistake_log.md`。
