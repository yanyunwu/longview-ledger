# 交易流水字段说明

| 字段 | 含义 |
|---|---|
| datetime | 操作发生时间 |
| asset_class | a股 / 港股 / 美股 / 贵金属 / 加密货币 |
| symbol | 代码或资产符号 |
| name | 标的名称 |
| action | buy / sell / add / reduce / clear / dividend / transfer_in / transfer_out / note |
| quantity | 数量 |
| price | 价格 |
| currency | 币种 |
| amount | 成交金额 |
| fee | 费用 |
| account | 账户 |
| position_pct | 操作后或计划仓位 |
| strategy_type | value_investment / cycle_allocation / small_position_speculation / risk_hedge / cash_management / watch_only |
| reason | 操作原因 |
| expected_outcome | 预期结果 |
| invalidation_condition | 失效条件 |
| source_message | 用户原始描述 |
| created_at | 记录创建时间 |
